FSRS Adaptive Learning Algorithm

Backed by research

The Free Spaced Repetition Scheduler (FSRS) tracks two parameters per concept per student: difficulty (how hard is this concept for this learner) and stability (how well is it retained over time). This creates a personalized review schedule that maximizes retention while minimizing study time.

Step 1

Initial Assessment

Student encounters a new concept. FSRS sets initial difficulty and stability based on first interaction.

Step 2

Adaptive Scheduling

Based on response quality, FSRS calculates optimal review interval. Harder concepts reviewed sooner.

Step 3

Memory Decay Model

Forgetting curve modeled per concept. Reviews scheduled just before predicted memory decay point.

Step 4

Continuous Optimization

Parameters updated after every interaction. The system gets more accurate the more the student uses it.
